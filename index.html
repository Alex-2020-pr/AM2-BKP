AM2 Backup Agent · Tecnologia de Ponta

Desenvolvemos nosso agente inteligente de backup para oferecer performance, segurança e flexibilidade onde seus dados realmente estão: em servidores físicos, virtuais ou estações críticas.

Um agente, múltiplas possibilidades

Multiplataforma: Windows, Linux, macOS – suporte nativo a qualquer ambiente.
Aplicações críticas: Agente homologado para Exchange, SQL Server, Oracle DB, MySQL, Hyper-V, VMware e Active Directory.
Deduplicação e compressão local: Redução de até 90% do tráfego e armazenamento.
Criptografia pré-envio: Dados são criptografados na origem com AES-256 – nem a AM2 consegue ler seu conteúdo.
Restauração granular: Recupere arquivos, pastas, objetos de banco de dados ou VMs inteiras com poucos cliques.

Como funciona:

  1. Agente instalado nos ativos protegidos
  2. Políticas configuradas por tipo de dado
  3. Dados criptografados e enviados à Oracle Cloud
  4. Gestão centralizada via console web AM2
  5. Restauração com poucos cliques, de qualquer lugar

Segurança Imutável · Anti-Ransomware

Criptografia AES-256 em repouso

Todo dado armazenado é criptografado com chaves gerenciadas pela AM2 ou pelo cliente (BYOK). Nem a Oracle tem acesso aos seus dados.

Comunicação TLS 1.2+ (mínimo)

Canais de backup e restauração exigem TLS 1.2 ou superior, garantindo que nenhum dado seja trafegado de forma insegura.

Proteção contra ransomware

Object storage com versionamento e imutabilidade (WORM). Uma vez escrito, nem administradores conseguem apagar ou modificar versões dentro do período de retenção.

Versionamento inteligente

Backup Full + Incremental forever com pontos de recuperação granulares. Mantenha versões de arquivos por meses ou anos sem overhead de storage.
